# Tempest
![screenshot_home](https://github.com/JohnathanCrow/tempest/blob/main/screenshot.png)

A metronome app with set list features, including per song double time toggle, frequency control, beat count, accents, practice speed modifiers, and notes.

## Installation

#### Access

- [Visit](https://johnathancrow.github.io/tempest/) and either bookmark or install via browser
- Data is stored locally


## Usage

#### Right Panel

- Add songs with the blue plus button
  - Each song has its own settings
  - Songs populate the right panel when added
- Add dividers with the - button
  - Dividers show in 'Custom' sorting view
- Import/Export/Clear
	- Import and export backups or separate set lists to file
  - Clear set list for a blank slate
- Sorting
  - Custom, alphabetical, BPM, capo position
  - Custom remembers order after using other sorting methods, drag and drop song to rearrange
  - Remove songs with the red 'x'
      
#### Left Panel

- Tempo display
  - The large number is the tempo display
  - Above indicates to modifiers applied (double time, speed options)
- Settings panel
  - Capo position, tempo, beats, value, frequency, and accent frequency options
  - Tap tempo button, double time toggle
- Controls
  - Previous, play/pause, next song
  - Accents control
  - Speed options for temporary adjustments
- Notes
  - Notes field for ideas, cues, or other information related to the current song
  - Songs with notes display a yellow dot in the right panel set list
